# Maze Ball — 傾けて転がす迷路 PWA

スマートフォンを傾けてボールを転がし、ランダム生成された迷路のゴールを目指す PWA ゲーム。
オフライン動作、難易度 1〜100、ハイスコア記録、ゴーストリプレイ、デイリーチャレンジ等を備えています。

## 特徴

| カテゴリ | 内容 |
|---|---|
| 難易度 | 1〜100。迷路サイズ・ボール直径・ループ数が連続的に変化 |
| トラップ穴 | 難易度 50 以上で出現。落ちるとスタートに戻ります |
| 視界制限 | 難易度 60 以上でフォグ・オブ・ウォー発動 |
| 鍵と扉 | 難易度 70 以上で鍵を取らないとゴールが開きません |
| カウントダウン | スタート前 3-2-1 で計測公平性を確保 |
| バイブ | 壁衝突・鍵取得・落下・ゴールで触覚フィードバック |
| サウンド | WebAudio で軽量 SE（壁衝突・鍵・穴落ち・ゴール） |
| 軌跡表示 | ボールの動きを残光として描画（ON/OFF 切替可） |
| ゴースト | 同じ迷路で再挑戦すると過去ベストが伴走 |
| シェイク | 端末を強く振ると即リスタート |
| デイリーチャレンジ | 日付シードで全プレイヤー共通の迷路 |
| キャリブレーション | 開始時の姿勢を「水平」基準に補正 |
| フォールバック | センサー不可なら矢印キー / 画面ジョイスティックで操作 |

## ファイル構成

```

│   ├── storage.js     # localStorage 管理

└── icons/
    ├── icon-192.png
    ├── icon-512.png
    └── icon-maskable-512.png
```

## ローカルでの動作確認

PWA は `https` または `localhost` でないと Service Worker が動きません。

```powershell
# Python があれば
python -m http.server 8080
# → http://localhost:8080/ を Chrome 等で開く
```

または Node.js の `npx serve` でも OK です。

## スマートフォンへのインストール

### iPhone / iPad (Safari)
1. Safari でページを開く
2. 共有ボタン（□↑）→ 「ホーム画面に追加」
3. ホーム画面のアイコンから起動するとフルスクリーン

### Android (Chrome)
1. Chrome でページを開く
2. ⋮ メニュー → 「アプリをインストール」
3. または、メニュー画面に表示される「📲 ホーム画面に追加」ボタン

## 操作

| デバイス | 操作 |
|---|---|
| iPhone (iOS 13+) | センサー使用許可 → 端末を傾けて操作 |
| Android | 端末を傾けて操作（許可ダイアログなし） |
| PC | 矢印キー / WASD（自動でフォールバックモード） |
| タッチデバイス | センサー拒否時は画面下部のジョイスティック |

シェイクでリスタート、HUD の ↻ ボタンでもリスタート可能です。

## 第三者のコード・素材について

このリポジトリは外部依存を持ちません（CDN・npm パッケージ・Web フォント等を一切利用していません）。アイコンは PowerShell の System.Drawing で自動生成したオリジナル画像です。フォントは利用者の OS 標準のもののみを参照しています。
